MOCCA HOME PAGE

Multiple OCCurences analysis

  • What Is Mocca?

    • Mocca is a repeat extractor. Given one or more sequences, and the approximate position of one of the repeats, Mocca will make it possible to extract all the other occurences and to test seamlessly for new bounds. Mocca will then generate a LOCAL multiple alignment with each sequence fragment identified by its coordinates.
