Getting the best out of T-Coffee




  • T-Coffee

    The following dataset embeds two known structures that have been aligned using a struture alignment package (ref_aln.aln). Using this reference alignment, try to produce with T-Coffee an alignment as accurate as possible.

    You can evaluate the accuracy of any T-Coffee generated alignment by comparing it with the reference, using the following command:
    t_coffee -other_pg aln_compare -al1 ref_aln.aln -al2 [your aln]
