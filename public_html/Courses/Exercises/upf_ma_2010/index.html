OBJECTIVES

This course is a primer on multiple sequence alignment methods. Its purpose is to show how multiple sequence alignments can be used, and what are the most basic algorithmic techniques for computing such alignments. All these tools will be presented in an evolutionnary context and several practical applications will be proposed. At the end of the course, students should know how to use the most basic sequence alignment methods while understanding their main limitations.
